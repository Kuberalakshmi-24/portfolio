Hallucination-Free RAG Pipeline Using BAAI Embeddings & MMR Retrieval

Architected a retrieval-augmented generation pipeline that eliminates hallucinations through BAAI/bge embeddings combined with Maximal Marginal Relevance (MMR) retrieval strategy. Evaluated rigorously using the RAGAS framework. Inference optimized via Groq LPU achieving sub-20ms end-to-end latency — a first-of-its-kind benchmark for production RAG systems.
